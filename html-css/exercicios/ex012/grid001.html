Desafio CSS Grid 001

Objetivo: Crie uma galeria de fotos em grade responsiva usando CSS Grid.

Instruções:

  • Crie 3 colunas iguais para as imagens
  • As imagens devem ocupar toda a largura da célula
  • Adicione gap de 10px entre as imagens
  • Teste redimensionando a janela (responsivo)
