Desafio Position 001

Objetivo: Crie um modal que aparece centralizado na tela quando clicar no botão.

Instruções:

  • Use position: relative no container pai
  • Use position: absolute/fixed no modal, centralizado
  • Crie backdrop fixed que cobre a tela
  • Adicione classes .active para mostrar/esconder (adicione JS simples se quiser)
